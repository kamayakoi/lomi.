## Obtenir de l’aide

### Support GitHub

Pour les questions techniques, les bugs et les demandes de fonctionnalités, le dépôt GitHub est le bon point de départ :

- **Questions** : ouvrez une [nouvelle issue](https://github.com/lomiafrica/lomi./issues/new?labels=question) avec l’étiquette `` `question` `` pour l’équipe et la communauté
- **Bugs** : un problème ou un dysfonctionnement ? [Signalez-le](https://github.com/lomiafrica/lomi./issues/new?labels=bug) avec l’étiquette `` `bug` `` ; nous examinerons le cas.
- **Fonctionnalités** : une idée ? [Partagez-la](https://github.com/lomiafrica/lomi./issues/new?labels=enhancement) avec l’étiquette `` `enhancement` ``.

### Support communauté

Rejoignez développeurs et marchands :

- **Documentation** : parcourez la [documentation](https://docs.lomi.africa/) pour les guides et références.
- **Discord** : échangez avec la communauté et l’équipe sur notre [serveur Discord](https://discord.gg/33syDfh9).
- **Contribution** : améliorez lomi. via des pull requests ou des issues sur [GitHub](https://github.com/lomiafrica/dashboard).

### Support direct

- **E-mail** : [hello@lomi.africa](mailto:hello@lomi.africa) pour le support ou les demandes business.
- **Appel** : réservez un [appel de 30 minutes](https://cal.com/babacar-diop-umkvq2/30min).

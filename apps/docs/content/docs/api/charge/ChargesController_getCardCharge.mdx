## Overview

Obtenir un encaissement carte

### When to use this

Use this endpoint when your flow needs `GET /charge/card/&#123;id&#125;` (Obtenir un encaissement carte).


## Authentication

Merchant routes require an API key in the `X-API-KEY` header (see [Integration overview](/reference/setup/overview)). Use a **test** key against `https://sandbox.api.lomi.africa` and a **live** key against `https://api.lomi.africa`.

Errors follow the standard JSON error format (status code and machine-readable message).

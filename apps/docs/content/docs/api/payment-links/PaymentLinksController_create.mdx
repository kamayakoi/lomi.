## Overview

Create a **shareable payment link**:

| `link_type` | What to send | Notes |
| --- | --- | --- |
| **`instant`** | `title`, `currency_code`, **`amount`** required for a fixed-price link | No `product_id` |
| **`product`** | `title`, `currency_code`, **`product_id`** (optionally `price_id` if the product has multiple prices) | Omit `amount` for product-driven pricing |

Optional UX flags: `allow_coupon_code`, `allow_quantity`, `require_billing_address`, `expires_at`, redirect URLs, `metadata`.

## Authentication

Merchant routes require an API key in the `X-API-KEY` header (see [Integration overview](/reference/setup/overview)). Use a **test** key against `https://sandbox.api.lomi.africa` and a **live** key against `https://api.lomi.africa`.

## Endpoint

`POST /payment-links`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

_No path parameters beyond the URL pattern._

### Query parameters

_No query parameters._

### Request body

Minimum required: `link_type`, `title`, `currency_code`.

**Instant link example**

```json
{
  "link_type": "instant",
  "title": "Invoice INV-001",
  "currency_code": "XOF",
  "amount": 15000,
  "success_url": "https://example.com/thanks"
}
```

**Product link example**

```json
{
  "link_type": "product",
  "title": "Premium plan",
  "currency_code": "XOF",
  "product_id": "123e4567-e89b-12d3-a456-426614174000"
}
```

## Responses

**201** Created — `PaymentLinkResponseDto` includes the public URL and identifiers.

| Status | Description |
| --- | --- |
| `201` | Lien de paiement créé avec succès |
| `400` | Entrée invalide ou erreur de validation |
| `401` | Clé API invalide ou manquante |

## Errors

**400** when mutually exclusive fields are combined (e.g. product link with amount-only rules) or enums are wrong—align with validation in `apps/api/src/core/payment-links`.

## Example

```bash
curl -sS -X POST "https://sandbox.api.lomi.africa/payment-links" \
  -H "X-API-KEY: $LOMI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "link_type":"instant",
    "title":"Invoice INV-001",
    "currency_code":"XOF",
    "amount":15000
  }'
```

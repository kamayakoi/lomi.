## Overview

Lancer un virement SPI

Initie un virement instantané SPI (RPC initiate_spi_payout). Le règlement PI-SPI se poursuit de façon asynchrone.


### When to use this

Use this endpoint when your flow needs `POST /payout/spi` (Lancer un virement SPI).


## Authentication

Merchant routes require an API key in the `X-API-KEY` header (see [Integration overview](/reference/setup/overview)). Use a **test** key against `https://sandbox.api.lomi.africa` and a **live** key against `https://api.lomi.africa`.

## Endpoint

`POST /payout/spi`

Base URLs:

- `https://sandbox.api.lomi.africa`

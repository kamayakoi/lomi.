## Overview

Returns one beneficiary payout by UUID, including current **status** and provider-facing fields. **404** if the id is unknown or not accessible with your key.

## Authentication

Merchant `X-API-KEY` — [Integration overview](/reference/setup/overview).

## Endpoint

`GET /beneficiary-payouts/{id}`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

| Name | Description |
| --- | --- |
| `id` | UUID du paiement |

### Query parameters

_None._

## Responses

| Status | Description |
| --- | --- |
| `200` | Détails du paiement |
| `401` | Clé API invalide ou manquante |
| `404` | Paiement introuvable ou accès refusé |

## Errors

**404** cross-tenant or unknown id.

## Example

```bash
curl -sS -X GET \
  "https://sandbox.api.lomi.africa/beneficiary-payouts/YOUR_PAYOUT_UUID" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

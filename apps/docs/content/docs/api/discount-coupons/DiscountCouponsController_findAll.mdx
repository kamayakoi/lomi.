## Overview

Returns **all discount coupons** for the merchant’s organization (no pagination in controller).

## Authentication

Merchant `X-API-KEY` — [Integration overview](/reference/setup/overview).

## Endpoint

`GET /discount-coupons`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

_None._

### Query parameters

_None._

## Responses

**200** — `DiscountCouponResponseDto[]`.

| Status | Description |
| --- | --- |
| `200` | Liste des coupons |
| `401` | Unauthorized |

## Errors

**401** invalid or missing key.

## Example

```bash
curl -sS -X GET "https://sandbox.api.lomi.africa/discount-coupons" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

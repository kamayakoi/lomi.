## Overview

Loads one coupon UUID. Returns **404** if the coupon doesn’t exist or isn’t tied to your organization key.

Note: **`GET /discount-coupons/:id/performance`** is a separate route; use that path for analytics, not `:id`.

## Authentication

Merchant `X-API-KEY`.

## Endpoint

`GET /discount-coupons/{id}`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

| Name |
| --- |
| `id` Coupon UUID |

### Query parameters

_None._

## Responses

| Status | Description |
| --- | --- |
| `200` | Détails du coupon |
| `401` | Unauthorized |
| `404` | Coupon introuvable ou accès refusé |

## Errors

**404** wrong tenant / unknown id.

## Example

```bash
curl -sS -X GET "https://sandbox.api.lomi.africa/discount-coupons/YOUR_COUPON_UUID" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

## Overview

Creates a coupon in your org. The **code is uppercased** server-side (per controller description).

**400** on invalid payload or **duplicate code** within org.

## Authentication

Merchant `X-API-KEY`.

## Endpoint

`POST /discount-coupons`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Request body

`CreateDiscountCouponDto` — see `apps/api/src/core/discount-coupons/dto/create-discount-coupon.dto.ts` and OpenAPI for required fields (code, discount type, limits, window, etc.).

## Responses

| Status | Description |
| --- | --- |
| `201` | Coupon créé |
| `400` | Données invalides ou code déjà utilisé |
| `401` | Unauthorized |

## Errors

**400** validation or uniqueness conflict.

## Example

```bash
curl -sS -X POST "https://sandbox.api.lomi.africa/discount-coupons" \
  -H "X-API-KEY: $LOMI_API_KEY" \
  -H "Content-Type: application/json" \
  -d @coupon-payload.json
```

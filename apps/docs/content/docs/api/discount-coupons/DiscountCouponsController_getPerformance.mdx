## Overview

Aggregate **usage & revenue metrics** for a single coupon UUID.

Response shape documented in Swagger: `total_uses`, `total_discount_amount`, `total_revenue`, `average_order_value`.

## Authentication

Merchant `X-API-KEY`.

## Endpoint

`GET /discount-coupons/{id}/performance`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

| Name |
| --- |
| `id` Coupon UUID |

### Query parameters

_None._

## Responses

**200** — metrics object (see controller `@ApiResponse` schema).

| Status | Description |
| --- | --- |
| `200` | Indicateurs |
| `401` | Unauthorized |
| `404` | Coupon introuvable ou accès refusé |

## Errors

**404** hidden resource.

## Example

```bash
curl -sS -X GET \
  "https://sandbox.api.lomi.africa/discount-coupons/YOUR_COUPON_UUID/performance" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

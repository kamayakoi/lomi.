Les soldes exposent la vue monétaire utile à une intégration marchande pour la réconciliation et la disponibilité avant virement. Ils remplacent l’ancien groupe public Accounts.

## Quand utiliser les soldes ?

Utilisez ces endpoints lorsque votre backend doit :

- Afficher les soldes disponibles et en attente.
- Vérifier qu’un montant est disponible avant virement.
- Réconcilier les transactions complétées avec les mouvements de solde.
- Construire des vues internes finance ou opérations.

Pour créer un virement, voir [Payouts](/api/payouts).

## Overview

Partially updates a webhook (rotate URL, toggle `is_active`, adjust encoded event subscriptions, verification token rotation, etc.). Only fields you send are changed.

Maps to [`UpdateWebhookDto`](https://github.com/lomiafrica/) — omit fields not changing.

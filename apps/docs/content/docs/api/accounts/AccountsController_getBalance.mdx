## Overview

Returns **current balance rows** for your organization. Omit `currency` to list all supported currencies on the account model, or pin a single currency (`XOF`, `USD`, `EUR`).

## Authentication

Merchant routes require an API key in the `X-API-KEY` header (see [Integration overview](/reference/setup/overview)). Use a **test** key against `https://sandbox.api.lomi.africa` and a **live** key against `https://api.lomi.africa`.

## Endpoint

`GET /accounts/balance`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

_None (static path `balance`)._

### Query parameters

| Name | Notes |
| --- | --- |
| `currency` | Optional. One of `XOF`, `USD`, `EUR` to filter balances to a single currency |

## Responses

**200** — array of `AccountBalanceResponseDto` (per-currency balance lines).

| Status | Description |
| --- | --- |
| `200` | Informations de solde |
| `401` | Clé API invalide ou manquante |

## Errors

**401** Unauthorized for bad key.

## Example

```bash
curl -sS -G "https://sandbox.api.lomi.africa/accounts/balance" \
  --data-urlencode "currency=XOF" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

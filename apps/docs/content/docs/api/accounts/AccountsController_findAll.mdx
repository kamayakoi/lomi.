## Overview

Returns **all settlement / balance accounts** tied to your authenticated organization (wallet-style accounts you use for payouts, holds, and reconciliation).

## Authentication

Merchant routes require an API key in the `X-API-KEY` header (see [Integration overview](/reference/setup/overview)). Use a **test** key against `https://sandbox.api.lomi.africa` and a **live** key against `https://api.lomi.africa`.

## Endpoint

`GET /accounts`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

_No path parameters._

### Query parameters

_None._

## Responses

**200** — array of `AccountResponseDto`.

| Status | Description |
| --- | --- |
| `200` | Liste des comptes |
| `401` | Clé API invalide ou manquante |

## Errors

**401** if the API key is missing or invalid for the environment.

## Example

```bash
curl -sS -X GET "https://sandbox.api.lomi.africa/accounts" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

## Overview

Returns **business metrics** for your organization: MRR, ARR, LTV, revenue, customer counts, etc. (see `OrganizationMetricsResponseDto` in OpenAPI).

This is a **static path** (`/metrics`) — not `GET /organizations/{id}`.

## Authentication

Merchant `X-API-KEY` — [Integration overview](/reference/setup/overview).

## Endpoint

`GET /organizations/metrics`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

_None._

### Query parameters

_None._

## Responses

| Status | Description |
| --- | --- |
| `200` | Indicateurs de l'organisation |
| `401` | Clé API invalide ou manquante |

## Errors

**401** bad key.

## Example

```bash
curl -sS -X GET "https://sandbox.api.lomi.africa/organizations/metrics" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

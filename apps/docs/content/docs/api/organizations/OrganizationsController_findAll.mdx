## Overview

Returns **organization profile(s)** for the authenticated merchant. The OpenAPI schema exposes an **array** of `OrganizationResponseDto` (same route as “current org” listing in the generated spec).

Use **`GET /organizations/metrics`** for KPIs, and **`GET /organizations/{id}`** when you need a specific org id.

## Authentication

Merchant `X-API-KEY` — [Integration overview](/reference/setup/overview).

## Endpoint

`GET /organizations`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

_None._

### Query parameters

_None._

## Responses

| Status | Description |
| --- | --- |
| `200` | Détails de l'organisation |
| `401` | Clé API invalide ou manquante |

## Errors

**401** bad or missing API key.

## Example

```bash
curl -sS -X GET "https://sandbox.api.lomi.africa/organizations" \
```

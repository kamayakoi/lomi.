## Overview

Paginated catalogue listing with embedded price rows (`ProductResponseDto` array returned by service).

`isActive` filter uses string parsing in controller: pass literal `"true"` or `"false"` to enable boolean filter; omit for all.

## Authentication

Merchant `X-API-KEY`.

## Endpoint

`GET /products`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Query parameters

| Param | Default | Notes |
| --- | --- | --- |
| `isActive` | n/a | `"true"`/`"false"` string |
| `limit` | `15` | Max rows |
| `offset` | `0` | Skip rows |

## Responses

| Status | Description |
| --- | --- |
| `200` | Product list |
| `401` | Unauthorized |

## Errors

**401** invalid or missing API key.

## Example

```bash
curl -sS -G "https://sandbox.api.lomi.africa/products" \
  --data-urlencode "isActive=true" \
  --data-urlencode "limit=30" \
  --data-urlencode "offset=0" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

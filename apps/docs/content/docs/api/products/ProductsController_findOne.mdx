## Overview

Loads one product with nested price objects. **404** if inaccessible.

## Authentication

Merchant `X-API-KEY`.

## Endpoint

`GET /products/{id}`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

| Name |
| --- |
| `id` product UUID |

### Query parameters

_None._

## Responses

| Status | Description |
| --- | --- |
| `200` | Détails |
| `401` | Unauthorized |
| `404` | Produit introuvable |

## Errors

**401** invalid key. **404** product absent or not visible for this merchant.

## Example

```bash
curl -sS -X GET "https://sandbox.api.lomi.africa/products/YOUR_PRODUCT_UUID" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

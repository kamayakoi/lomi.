## Overview

Attach a new **price option** to existing product `id`. Existing prices are immutable—create a fresh row for plan changes.

Hard cap: controller documents **max 3 active prices** per product; exceeding returns **400**.

## Authentication

Merchant `X-API-KEY`.

## Endpoint

`POST /products/{id}/prices`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

| Name |
| --- |
| `id` product UUID |

### Request body

`AddPriceDto` (amount, currency cadence, nickname, etc.—see DTO file).

## Responses

| Status | Description |
| --- | --- |
| `201` | Price created |
| `400` | Validation / max prices |
| `401` | Unauthorized |
| `404` | Product introuvable |

## Errors

**400** DTO validation or **max 3 active prices** exceeded. **401** unauthorized. **404** product not found for this org.

## Example

```bash
curl -sS -X POST \
  "https://sandbox.api.lomi.africa/products/YOUR_PRODUCT_UUID/prices" \
  -H "X-API-KEY: $LOMI_API_KEY" \
  -H "Content-Type: application/json" \
  -d @add-price.json
```

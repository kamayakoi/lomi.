## Overview

Sets which **price UUID** is chosen when buyers do not explicitly pass `price_id`. Only one default at a time — new default supersedes prior.

## Authentication

Merchant `X-API-KEY`.

## Endpoint

`POST /products/{id}/prices/{priceId}/set-default`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

| Name | Role |
| --- | --- |
| `id` | Product UUID |
| `priceId` | Price row UUID |

_No JSON body on this route._

## Responses

**200** refreshed `ProductResponseDto` with default flag updated.

| Status | Description |
| --- | --- |
| `200` | Default updated |
| `401` | Unauthorized |
| `404` | Product or price missing |

## Errors

**401** invalid key. **404** product/price missing or price not attached to that product.

## Example

```bash
curl -sS -X POST \
  "https://sandbox.api.lomi.africa/products/PROD_UUID/prices/PRICE_UUID/set-default" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

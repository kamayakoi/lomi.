## Overview

Returns a single **delivery log** row by UUID (status, HTTP result, timing). **404** if the log does not exist or is not visible for your API key.

## Authentication

Merchant `X-API-KEY` — [Integration overview](/reference/setup/overview).

## Endpoint

`GET /webhook-delivery-logs/{id}`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

| Name | Description |
| --- | --- |
| `id` | UUID du journal |

### Query parameters

_None._

## Responses

| Status | Description |
| --- | --- |
| `200` | Détails du journal de livraison |
| `401` | Clé API invalide ou manquante |
| `404` | Journal introuvable ou accès refusé |

## Errors

**404** wrong id or other merchant’s log.

## Example

```bash
curl -sS -X GET \
  "https://sandbox.api.lomi.africa/webhook-delivery-logs/YOUR_LOG_UUID" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

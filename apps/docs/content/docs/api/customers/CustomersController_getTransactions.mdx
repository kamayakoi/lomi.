## Overview

Loads **transactions associated with customer `id`** (payment history auditing). Nested route **must** declare `:id/transactions`.

## Authentication

Merchant `X-API-KEY`.

## Endpoint

`GET /customers/{id}/transactions`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

| Name |
| --- |
| `id` (customer UUID) |

### Query parameters

_None in controller._

## Responses

**200** — `TransactionResponseDto[]`

| Status | Description |
| --- | --- |
| `200` | Liste transactions |
| `401` | Unauthorized |
| `404` | Client introuvable ou accès refusé |

## Errors

Customer must exist beforehand.

## Example

```bash
curl -sS -X GET "https://sandbox.api.lomi.africa/customers/YOUR_UUID/transactions" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

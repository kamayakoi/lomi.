## Overview

Soft-removes customer visibility per API contract description: customer **no longer appears** in listings or fetch-by-id—align your CRM expectations accordingly.

## Authentication

Merchant `X-API-KEY`. See [Integration overview](/reference/setup/overview).

## Endpoint

`DELETE /customers/{id}`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

| Name | Required |
| --- | --- |
| `id` | Customer UUID |

### Query parameters

_None._

## Responses

| Status | Description |
| --- | --- |
| `200` | `{ "message": "Client supprimé avec succès" }` style payload |
| `401` | Unauthorized |
| `404` | Client introuvable ou accès refusé |

## Errors

**404** if not found / other org.

## Example

```bash
curl -sS -X DELETE "https://sandbox.api.lomi.africa/customers/YOUR_UUID" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

## Overview

Fetch one customer UUID. Hidden from other merchants — **404** if not visible to calling key.

## Authentication

Merchant routes require an API key in the `X-API-KEY` header (see [Integration overview](/reference/setup/overview)).

## Endpoint

`GET /customers/{id}`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

| Name | Required |
| --- | --- |
| `id` | UUID client |

### Query parameters

_None._

## Responses

| Status | Description |
| --- | --- |
| `200` | Détails du client |
| `401` | Unauthorized |
| `404` | Client introuvable ou accès refusé |

## Errors

**404** cross-tenant or unknown id.

## Example

```bash
curl -sS -X GET "https://sandbox.api.lomi.africa/customers/YOUR_CUSTOMER_UUID" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

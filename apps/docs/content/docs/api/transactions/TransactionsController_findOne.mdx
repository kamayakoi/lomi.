## Overview

Returns a single **transaction** by UUID. Responds **404** if the id does not exist or is not visible for your API key (cross-organization access is not exposed).

## Authentication

Merchant `X-API-KEY` — [Integration overview](/reference/setup/overview).

## Endpoint

`GET /transactions/{id}`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

| Name | Description |
| --- | --- |
| `id` | UUID de la transaction |

### Query parameters

_None._

## Responses

| Status | Description |
| --- | --- |
| `200` | Détails de la transaction |
| `401` | Clé API invalide ou manquante |
| `404` | Transaction introuvable ou accès refusé |

## Errors

**401** auth failure. **404** wrong id or not your organization.

## Example

```bash
curl -sS -X GET "https://sandbox.api.lomi.africa/transactions/YOUR_TRANSACTION_UUID" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

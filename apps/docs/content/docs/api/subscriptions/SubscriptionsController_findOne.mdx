## Overview

Single subscription detail (`SubscriptionResponseDto`).

## Authentication

Merchant `X-API-KEY`.

## Endpoint

`GET /subscriptions/{id}`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

| Name | Description |
| --- | --- |
| `id` | Subscription UUID |

### Query parameters

_None._

## Responses

| Status | Description |
| --- | --- |
| `200` | Détails |
| `401` | Unauthorized |
| `404` | Abonnement introuvable ou accès refusé |

## Errors

**404** hidden cross-tenant resource.

## Example

```bash
curl -sS -X GET "https://sandbox.api.lomi.africa/subscriptions/YOUR_SUB_UUID" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

## Overview

Lists subscriptions auto-created when customers purchase recurring products. Paginate with `page` + `pageSize`.

## Authentication

Merchant `X-API-KEY` — [Integration overview](/reference/setup/overview).

## Endpoint

`GET /subscriptions`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

_None._

### Query parameters

| Name | Default | Role |
| --- | --- | --- |
| `page` | `1` | Pagination index |
| `pageSize` | `50` | Page size |

## Responses

**200** — `SubscriptionResponseDto[]`.

| Status | Description |
| --- | --- |
| `200` | Liste des abonnements |
| `401` | Unauthorized |

## Errors

**401** invalid key.

## Example

```bash
curl -sS -G "https://sandbox.api.lomi.africa/subscriptions" \
  --data-urlencode "page=1" \
  --data-urlencode "pageSize=25" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

Subscriptions cover recurring billing created through checkout, payment links, or backend workflows. Customer-scoped subscription operations live here too, so the API reference has one place for recurring billing.

## What belongs here

- List and retrieve subscription instances.
- Update subscription metadata or supported fields.
- Cancel a subscription.
- Inspect subscriptions for a specific customer.
- Use customer-scoped operations when your customer portal or admin tools need them.

Les abonnements couvrent la facturation récurrente créée via checkout, liens de paiement ou workflows backend. Les opérations par client vivent aussi ici afin d’avoir une seule référence pour la facturation récurrente.

## Ce qui appartient à cette section

- Lister et récupérer les instances d’abonnement.
- Mettre à jour les métadonnées ou champs supportés.
- Résilier un abonnement.
- Inspecter les abonnements d’un client spécifique.
- Utiliser les opérations par client pour le portail client ou les outils admin.

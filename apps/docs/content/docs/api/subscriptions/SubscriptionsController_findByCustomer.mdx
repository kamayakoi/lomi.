## Overview

List subscriptions for customer

Returns subscriptions tied to one customer ID. Responds with **404** when the customer is unknown.


### When to use this

Use on customer portals showing active plans.

### See also

[Retrieve customer](/api/customers/CustomersController_findOne)


## Authentication

Merchant routes require an API key in the `X-API-KEY` header (see [Integration overview](/reference/setup/overview)). Use a **test** key against `https://sandbox.api.lomi.africa` and a **live** key against `https://api.lomi.africa`.

## Endpoint

`GET /subscriptions/customer/{customerId}`

## Overview

Scoped listing of **subscriptions for `customerId`** UUID (must belong to your org). Returns **404** customer missing / inaccessible.

## Authentication

Merchant `X-API-KEY`.

## Endpoint

`GET /subscriptions/customer/{customerId}`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

| Name |
| --- |
| `customerId` |

### Query parameters

_None._

## Responses

**200** — `SubscriptionResponseDto[]`.

| Status | Description |
| --- | --- |
| `200` | Liste des abonnements du client |
| `401` | Unauthorized |
| `404` | Client introuvable ou accès refusé |

## Errors

**404** cross-tenant lookup.

## Example

```bash
curl -sS -X GET \
  "https://sandbox.api.lomi.africa/subscriptions/customer/YOUR_CUSTOMER_UUID" \
  -H "X-API-KEY: $LOMI_API_KEY"
```

## Overview

**Cancel** an active subscription—the **only** mutable operation exposed on subscriptions in this API.

Optional JSON body with `cancellation_reason` captured for analytics.

## Authentication

Merchant `X-API-KEY`.

## Endpoint

`POST /subscriptions/{id}/cancel`

Base URLs:

- `https://sandbox.api.lomi.africa`
- `https://api.lomi.africa`

## Request

### Path parameters

| Name |
| --- |
| `id` |

### Query parameters

_None._

### Request body

```json
{
  "cancellation_reason": "Demande du client"
}
```

## Responses

| Status | Description |
| --- | --- |
| `200` | Abonnement résilié avec succès |
| `401` | Unauthorized |
| `404` | Abonnement introuvable ou accès refusé |

## Errors

**404** not found or wrong organization.

## Example

```bash
curl -sS -X POST \
  "https://sandbox.api.lomi.africa/subscriptions/YOUR_SUB_UUID/cancel" \
  -H "X-API-KEY: $LOMI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"cancellation_reason":"Cancelled by user"}'
```

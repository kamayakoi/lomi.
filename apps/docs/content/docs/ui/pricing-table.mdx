---
title: Pricing Table
description: A responsive pricing table for subscription tiers with a monthly/yearly toggle.
---

The `PricingTable` is a standalone, copy-pasteable UI component built with standard HTML elements and Tailwind CSS semantic utility classes. It adapts automatically to your theme without requiring additional Shadcn primitives.

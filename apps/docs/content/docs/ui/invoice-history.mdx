---
title: Invoice History
description: A responsive data table component for displaying a customer's past invoices.
---

The `InvoiceHistory` is a standalone, copy-pasteable UI component built with standard HTML elements and Tailwind CSS semantic utility classes. It adapts automatically to your theme without requiring additional Shadcn primitives.

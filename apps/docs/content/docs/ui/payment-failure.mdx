---
title: Payment Failure Card
description: A component to display when a payment fails, with options to retry or contact support.
---

The `PaymentFailureCard` is a standalone, copy-pasteable UI component built with standard HTML elements and Tailwind CSS semantic utility classes. It adapts automatically to your theme (light/dark mode) and primary colors without requiring you to install additional Shadcn primitives.

## Props

| Prop | Type | Description |
| ---- | ---- | ----------- |
| `title` | `string` | The title of the failure card. |
| `description` | `string` | A description of what went wrong. |
| `errorDetails` | `string` | Optional technical error details. |
| `onRetry` | `() => void` | Callback when the user clicks "Try again". |
| `onContactSupport` | `() => void` | Callback when the user clicks "Contact support". |
| `className` | `string` | Optional extra class names. |

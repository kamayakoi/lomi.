---
title: Checkout Summary Card
description: A component displaying the summary of a checkout order, including line items, subtotal, and total.
---

The `CheckoutSummaryCard` is a standalone, copy-pasteable UI component built with standard HTML elements and Tailwind CSS semantic utility classes. It adapts automatically to your theme without requiring additional Shadcn primitives.

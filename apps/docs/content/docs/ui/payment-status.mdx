---
title: Payment Status Card
description: A component to communicate the status of a payment (processing, success, failed, etc).
---

The `PaymentStatusCard` is a standalone, copy-pasteable UI component built with standard HTML elements and Tailwind CSS semantic utility classes. It adapts automatically to your theme without requiring additional Shadcn primitives.

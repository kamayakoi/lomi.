## Clés API

lomi. propose deux types de clés API pour chaque environnement :

| Clé | Préfixe (actuel) | Usage |
| --- | --- | --- |
| **Secrète** | `lomi_sk_test_…` / `lomi_sk_live_…` | Serveurs, scripts, CI — accès API complet. **Jamais dans le navigateur ou une app mobile.** |
| **Publiable** | `lomi_pk_test_…` / `lomi_pk_live_…` | Confirmation carte côté client, Payment Elements — opérations limitées uniquement. |

<Callout type="warn">
  **Ne mettez jamais de clé secrète dans le frontend.** Utilisez les clés publiables seulement là où le guide d’intégration l’autorise explicitement.
</Callout>

<Callout type="info">
  Les anciens exemples peuvent afficher les préfixes `sk_test_` / `sk_live_`. Les clés actuellement délivrées par le tableau de bord suivent **`lomi_sk_test_`** et **`lomi_sk_live_`**, comme dans [Authentification](/reference/setup/authentication).
</Callout>

## Gestion des variables d’environnement

Créez un fichier `.env` à la racine du projet (ajoutez `.env` au `.gitignore`). Utilisez des clés **test** pour le bac à sable (`https://sandbox.api.lomi.africa`).

```bash filename=".env.example"
# Bac à sable (recommandé pendant l’intégration)
LOMI_SECRET_KEY=lomi_sk_test_xxxxxxxxxxxxxxxxxxxxxx
LOMI_PUBLISHABLE_KEY=lomi_pk_test_xxxxxxxxxxxxxxxxxxxxxx

# Secret de signature webhook (Développeurs → Webhooks → votre point de terminaison)
LOMI_WEBHOOK_SECRET=whsec_xxxxxxxxxxxxxxxxxxxxxx

# Optionnel : si vous appelez l’API sans SDK
# LOMI_API_URL=https://sandbox.api.lomi.africa
```

Copiez vers `.env` et remplacez les valeurs depuis **Paramètres → Jetons d’accès** et la configuration du webhook. Chargez `.env` avec votre framework (`dotenv`, Next.js, Vite, etc.) — ne commitez pas de vraies clés.

## Vérification des webhooks

Vérifiez les webhooks entrants à l’aide de `LOMI_WEBHOOK_SECRET` :

<Tabs items={["TypeScript", "Python", "Go", "PHP"]}>
<Tab value="TypeScript">
```typescript
// Express example
import crypto from 'crypto';

app.post('/webhook', (req, res) => {
const signature = req.headers['x-lomi-signature'];
const secret = process.env.LOMI_WEBHOOK_SECRET!;

const expected = crypto
.createHmac('sha256', secret)
.update(req.rawBody)
.digest('hex');

if (signature !== expected) {
return res.status(400).send('Invalid signature');
}

// Process event...
});

````
</Tab>
<Tab value="Python">
```python
# Flask example
import hmac
import hashlib

@app.route('/webhook', methods=['POST'])
def webhook():
    signature = request.headers.get('x-lomi-signature')
    secret = os.environ['LOMI_WEBHOOK_SECRET'].encode()

    expected = hmac.new(secret, request.data, hashlib.sha256).hexdigest()

    if not hmac.compare_digest(signature, expected):
        return jsonify({'error': 'Invalid signature'}), 400

    # Process event...
````

</Tab>
<Tab value="Go">
```go
func webhookHandler(w http.ResponseWriter, r *http.Request) {
    signature := r.Header.Get("x-lomi-signature")
    secret := os.Getenv("LOMI_WEBHOOK_SECRET")
    
    body, _ := io.ReadAll(r.Body)
    mac := hmac.New(sha256.New, []byte(secret))
    mac.Write(body)
    expected := hex.EncodeToString(mac.Sum(nil))
    
    if !hmac.Equal([]byte(signature), []byte(expected)) {
        http.Error(w, "Invalid signature", http.StatusBadRequest)
        return
    }
}
```
</Tab>

La **CLI lomi.** est un outil en ligne de commande pour travailler avec la plateforme de paiement lomi. depuis le terminal : authentification, génération de projets SDK, serveur webhook local, sessions de checkout, liens de paiement et règles pour Cursor, Claude et autres assistants IA.

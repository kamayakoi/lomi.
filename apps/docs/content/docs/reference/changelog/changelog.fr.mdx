## Version 1

- **Version initiale :** première version de notre API REST (sans préfixe de version global dans les chemins URL).
- **Ressources principales :** endpoints pour notamment :
  - [Authentification](/reference/setup/authentication)
  - [Fournisseurs](/reference/reference/data-models#provider-object)
  - [Webhooks](/reference/payments/webhooks)
  - [Données marchand](/reference/platform/merchants)
  - [Clients](/reference/platform/customers)
  - [Plans](/reference/payments/subscriptions)
  - [Produits](/reference/payments/products)
  - [Sessions de paiement](/reference/payments/checkout-sessions)
  - [Abonnements clients](/reference/platform/customer-subscriptions)
  - [Liens de paiement](/reference/payments/payment-links)
  - [Transactions](/reference/payments/transactions)
  - [Remboursements](/reference/payments/refunds) (bêta)
- **Environnements :** séparation Test et Live.
- **Gestion d’erreurs :** structure d’erreur normalisée.
- **CLI lomi.** : première version de l’interface `lomi.cli`.
- **SDK Node.js** : première version du paquet `@lomi./sdk`.
- **Payez ce que vous voulez :** le tunnel hébergé accepte des montants unitaires personnalisés dans les bornes configurées. Les tarifs produits acceptent `pricing_model`, `minimum_amount` et `maximum_amount` ; la validation du montant de session respecte les règles PWYW. Voir [Produits](/reference/payments/products) et [Comportement du tunnel](/reference/payments/checkout-behavior).

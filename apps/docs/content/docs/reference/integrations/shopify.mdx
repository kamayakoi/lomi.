# Shopify

<Callout type="warn">
  **Work in progress.** Merchant-facing setup for Lomi on Shopify is not finished in these docs. The codebase started from the official **Shopify App Template (Remix)** and adds Supabase-backed session storage and Lomi-related configuration — behavior and operational guidance may change.
</Callout>

## Current scope

The app in [`apps/plugins/shopify`](https://github.com/lomiafrica/lomi./tree/master/apps/plugins/shopify) focuses on:

- **Shopify OAuth** and embedded admin via `@shopify/shopify-app-remix`
- **Supabase** for Shopify session storage and merchant-specific Lomi settings (see README **Application Storage**)
- Standard Remix / Shopify CLI **local dev** and deployment flows

It does **not** yet mirror the same **hosted checkout session + dashboard webhook** narrative documented for WooCommerce, Magento, and PrestaShop in [Platform plugins integration contract](https://github.com/lomiafrica/lomi./blob/master/apps/plugins/INTEGRATION_CONTRACT.md).
